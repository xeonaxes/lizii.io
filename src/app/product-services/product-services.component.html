О проекте
Платформа по доставке готовых блюд из ресторанов и кафе

Перед нами стояла задача создать комплексный продукт, который позволял бы осуществлять доставку готовой еды из заведений общественного питания, при этом удовлетворяя всех пользователей данного продукта. Таких было четверо – конечный клиент, курьер, ресторан, сотрудники платформы.

Поэтому нами были продуманы и разработаны следующие модули:

Web/iOS/Android - для клиента;

Web/iOS/Android - для ресторана.

Web - для сотрудников платформы;

Android - для курьеров.
